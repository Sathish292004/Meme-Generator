---

## 🚀 Live Demo

🔗 https://meme-generator-eta-eight.vercel.app/

---

## 📖 Overview

Meme Generator is a web application that allows users to create hilarious memes by adding custom captions to popular meme templates. Users can search memes, customize text, preview changes in real time, and export memes in various formats.

---

## ✨ Features

* 😂 Generate memes instantly
* 🔍 Search meme templates
* ✏️ Add custom top and bottom text
* 🎨 Real-time meme preview
* 🌙 Dark & Light Mode
* 📜 Meme History Tracking
* 📤 Download memes
* ⚡ Fast React + Vite performance
* 🤖 Dynamic AI Meme Generation
* 📱 Responsive Design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sathish292004/Meme-Generator.git
```

### Navigate to Project

```bash
cd Meme-Generator
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Run API Server

```bash
npm run dev:api
```

### Run Full Application

```bash
npm run dev:full
```
